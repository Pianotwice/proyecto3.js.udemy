# proyecto3.js.udemy
Una página pequeñita con algunos usos de algunos elementos que aprendí en JavaScript en este largo proceso de aprendizaje, la idea es hacer click en COMENZAR y responder a 5 preguntas dentro un máximo de tiempo de 30 segundos. No se preocupen, son las preguntas más fáciles de la vida, incluso mi bebé de 7 meses las responde.

La idea es poder ejercitar elementos propios del curso de JavasCript comprado en UDEMY. Mi ambición a la larga es poder crear un juego que cautive a las próximas generaciones, donde la gente pueda hacer su vida en la web, pero hoy por hoy vamos quemando etsapas, aprendiendo JAVA, y en este caso aprendiendo JAVASCRIPT.

No me destruyan por la calidad de los estilos, que aun estoy aprendiendo. Muchas gracias.
